# Stream2Chat
## Development in Progress

Steps to run:
<br />
1. Clone the project:
   ```
   git clone https://github.com/hammoh7/Stream2Chat.git
   ```
2. Run the command:
   ```
   npm install
   ```
3. Then, run the development server:
   <br />
   ```
   cd frontend
   npm run dev
   ```
   ```
   cd backend
   npm run start
   ```
4. Open (http://localhost:5173) with your browser to see the result.
